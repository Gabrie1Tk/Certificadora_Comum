Exercícios Práticos - Aula 01

HTML e Semântica

Exercício 1: Navegação com Âncoras

Cria um menu de navegação para a tua página utilizando a tag semântica correta. Cria uma lista não ordenada com pelo menos três links (Ex: Início, Catálogo, Contato) que apontem para IDs dentro da própria página (ex: href="#catalogo").

Exercício 2: Estrutura Semântica do Cartão

Vamos estruturar o cartão de um carro. Utiliza a tag ideal para conteúdos independentes (article) para englobar o cartão. Insere uma imagem, um título (nome do carro), parágrafos para marca, ano e preço, e um botão de "Favoritar".

Exercício 3: Acessibilidade na Prática

Aprimora o HTML do cartão criado. Adiciona o atributo alt na imagem do carro para leitores de ecrã e o atributo aria-label no botão de "Favoritar", descrevendo a ação do ícone.
