# Exercícios Práticos - Aula 01: HTML e Semântica

Chegou a hora de colocar a mão na massa! Estes três exercícios vão consolidar o que aprendemos sobre estrutura, semântica e acessibilidade na web.

**Objetivo:** Criar um arquivo HTML simples contendo a estrutura de navegação e um cartão de carro.

### Exercício 1: Navegação com Âncoras
Crie um menu de navegação para a sua página.
* Utilize a tag semântica correta para agrupar os links de navegação.
* Crie uma lista não ordenada com pelo menos três links (Ex: Início, Catálogo, Contato).
* Os links devem ser "âncoras", ou seja, apontar para IDs dentro da própria página (ex: `href="#catalogo"`).

### Exercício 2: Estrutura Semântica do Cartão
Vamos estruturar o cartão de um carro (como o Civic ou o Corolla do nosso catálogo).
* Utilize a tag ideal para conteúdos independentes e auto-contidos para englobar o cartão.
* Dentro do cartão, insira uma imagem, um título (para o nome do carro), um parágrafo (para a marca e o ano) e um outro parágrafo com o preço.
* Inclua também um botão para a ação de "Favoritar".

### Exercício 3: Acessibilidade na Prática
Aprimore o HTML do cartão criado no Exercício 2 pensando em acessibilidade.
* Adicione o atributo correto na imagem do carro para que leitores de tela possam descrevê-la caso a imagem não carregue ou para usuários com deficiência visual.
* O botão de "Favoritar" muitas vezes usa apenas um ícone. Como vimos na aula, adicione o atributo que permite ao leitor de tela ler a ação correta desse botão invisível.
